# 🔍 Buscador de CEP

Aplicação web que consulta endereços a partir de um CEP brasileiro,
utilizando a API pública do ViaCEP.

## 🚀 Sobre o projeto

Este projeto foi desenvolvido para praticar:

- Consumo de API REST com `fetch` e `async/await`
- Manipulação do DOM
- Validação de formulário
- Tratamento de erros
- Estrutura semântica com HTML5

## 📌 Funcionalidades

✔ Busca de endereço pelo CEP  
✔ Validação de CEP incompleto (menos de 8 dígitos)  
✔ Tratamento de CEP inexistente  
✔ Exibição de CEP, rua, bairro, localidade e estado  
✔ Limpeza automática dos campos a cada nova busca  

## 🛠 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- [ViaCEP API](https://viacep.com.br/)

## 🎯 Como usar

1. Digite um CEP válido com 8 dígitos no campo de busca
2. Clique em **Encontrar**
3. O endereço completo será exibido na tela

## 💡 Próximas melhorias

- Máscara de CEP no input (00000-000)
